Kompakt

by Jasper ter Weeme

version 0.1.1

Installation
------------
Just install KOMPAKT.EXE somewhere and make sure CWSDPMI.EXE is
in your PATH.

Usage
-----
kompakt [options]
Options:
    -h Print help
    -x Extract
    -o followed by output directory
    -f followed by input ISO file
    -s Input from stdin
    -l List files
    -d Show descriptors
    -p Show path table
    -v verbose
